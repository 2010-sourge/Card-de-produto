# 🛒 Product Card

Um card de produto moderno e responsivo para e-commerce, exibindo informações de uma câmera **Sony Alpha A7 Kit** com descrição, avaliação por estrelas, preço com desconto e botão de compra.

---

## 📸 Preview

> Card com duas seções: à esquerda, descrição do produto, avaliação em estrelas, preço original riscado e preço promocional com botão "Add To Cart". À direita, imagem da câmera com fundo rosa/magenta, nome do produto e botão "Login For More".

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/product-card.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd product-card
   ```

3. Abra o `index.html` no navegador, **ou** use o **Live Server** do VS Code:
   - Instale a extensão **Live Server**
   - Clique com o botão direito em `index.html` → **Open with Live Server**
   - Acesse em: `http://127.0.0.1:5500`

---

## 🎯 Funcionalidades

- Card de produto dividido em duas colunas
- Seção de descrição com texto, avaliação por estrelas e preços
- Preço original riscado (`$1800.00`) e preço promocional em destaque (`$1500.00`)
- Botão **Add To Cart** em azul ciano
- Seção de imagem com fundo rosa/magenta e nome do produto (**Sony Alpha A7 Kit**)
- Tag **Free Shipping** (frete grátis)
- Botão **Login For More** para acesso a mais detalhes

---

## 🎨 Design

- Fundo da página: cinza claro com textura pontilhada
- Card: fundo branco/creme com sombra suave
- Seção de imagem: fundo rosa/magenta com texto em branco
- Preço promocional: vermelho/rosa em destaque
- Tipografia: bold para títulos, regular para descrição
